HTML Formatting Elements

Examples of Formatting Tags

Bold text: This text is bold using the <b> tag.

Important text: This text is important using the <strong> tag.

Italic text: This text is italicized using the <i> tag.

Emphasized text: This text is emphasized using the <em> tag.

Marked text: This text is marked using the <mark> tag.

Smaller text: This text is smaller using the <small> tag.

Deleted text: This text is deleted using the <del> tag.

Inserted text: This text is inserted using the <ins> tag.

Subscript text: H2O is water.

Superscript text: E = mc2 is Einstein's equation.
